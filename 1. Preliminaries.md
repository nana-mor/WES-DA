# 1.3 Essential Python Libraries

**Numpy**: One of its primary uses in data analysis is as a container for data to be passed between algorithms and libraries. Also, libraries written in a lower-level language, such as C o r FORTRAN, can operate on the data stored in a NumPY array without copying data into some other memory representation. 

**Pandas**: Provides high-level data structures and functional to make working with data intuitive and flexible. It has the following primary objects:
- DataFrame: tabular, column-oriented data structure with both row and column labels.
- Series: a one-dimensional labeled array object.
It blends the array-computing ideas of Numpy with the data manipulation capabilities found in spreadsheets and relational databases.
Also, many features found in pandas are typically either part of the R core implementation or provided by add-on packages.
The name is derived from *panel data*, a term for multidimensional structured datasets.

**Matplotlib**: Produces plots and other two-dimensional data visualizations. It is designed for creating plots suitable for publications. 

**SciPi**: Collection of packages addressing a number of foundational problems in scientific computing. For example, integrate, linalg, optimize, signal, sparse, special, stats, among others.

**Scikit-learn**: it includes sub-modules for machine learning models for Python. Is more focused on predictions.

**statsmodels**: a number of regression analysis models popular in the R programming language. It contains algorithms for classical (primarily frequentist) statistics and econometrics. Is more focused on statistical inference.
