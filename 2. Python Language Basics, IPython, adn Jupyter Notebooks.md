```Python

sep 
	string inserted between values, default a space. 
end

file
```

# 2.3 Python Language Basics

## Language Semantics

### Everything is an object

An importan characteristic of the Python language is the consistency of its *object model*. Everything you can find in a Python code is a *Python object*. Each object has an associated type (e.g., integer, string or function) and internal data.

### Comments
Any text preceded by the hash mark (pound sign) `#` is ignored by the Python interpreter.
```Python
print("Reached this line") # Simple status report
```

### Variables and argument passing

When assigning a variable you are creating a *reference* to the object shown on the righthand side of the equals sign. For example:

```Python
a = [1,2,3,4]
b = a
b
```
Creates the output:
```Output
[1, 2, 3, 4]
```
Because in this case `a`and `b` now refer to the same object.
```mermaid
flowchart LR
	A[a] --> C{"List [1, 2, 3, 4]"}
	B[b] --> C{"List [1, 2, 3, 4]"}
```

> [!info] Nota
> Assignment is also referred to as *binding*, as we are binding a name to an object.

When you pass objects as arguments to a function, new local variables are created referencing the original objects without any copying. For example, the code:

```Python
def append_element(list, element):
    list.append(element)
append_element(a, 5) #There is no other assignment to a
a
```
Creates the output:
```Shell
[1, 2, 3, 4, 5] 
```
### Dynamic references, strong types

A variable can refer to a different type of object simply by doing an assignment. Variables are names for objects within a particular namespace; the type information is stored in the object itself. 

In Python, some implicit casts are not allowed, this is why Python is a *strongly typed* language, wichh means that every object has a specific type (or class).

#### isinstance

You can use it to check that an object is an instance of a particular type. For example, the following code:

```Python
a = 5
isinstance(a,int)
```

Produces the output 
```shell
True
```

### Attributes and methods

Objects in Python typically have both attributes and methods.

Attribute: Other Python objects stored "inside" the object
Method: Functions associated with an object that can have access to the object's internal data.

#### getattr 

Atributes and methods can also be accessed by name via `getattr` function. For example the following code:

```Python
name = "Dana"
getattr(name, "split")
```

Produces the output
```shell
<function str.split(sep=None, maxsplit=-1)>
```
### Duck Typing
Often you may only care whether an object has certain methods or behavior instead of knowing the type of it.

#### iter

The `iter` function can be used to build a function that returns whether if an object is iterable or not. For example, the following code:
```Python
def isiterable(obj):
    try:
        iter(obj)
        return True
    except TypeError: #not iterable
        return False
    
isiterable("a string")
```

Produces the output

```shell
True
```

### Imports
In Python, a *module* is a file tih the `.py` extension containing Python code. If we eanted to access the variables and functions defined in another module, from another file in the same directory we could do:

```Python
import some_module #.py extension is not explicit
result = some_module.f(5)
```

#### as
Using the `as` keyword you can give imports different variable names:

```Python
import some_module as sm #.py extension is not explicit
result = sm.f(5)
```

### Binary operators and comparisons

#### is keyword

It is used to check if two variables refer to the same object. It follows the same logic if you need to check if a variable refers to `None` object. For instance, the following code:

```Python
a = [1,2,3]
b = a
c = list(a)
d = None
print(a is b)
print(a is c)
print(a is d)
print(d is None)
print(a is None)
```
Produces the output:
```shell
True
False
False
True
False
```
Since the `list` function always creates a new Python list (i.e., a copy), `c` refers to a distinct object than `a`. Comparing with `is` is not the same as the `==` operator. In this case, using the same variables as above, we have:
```Python
a == c
```
Produces the output:
```shell
True
```

Examples of mutable objects in Python:
- Lists
- Dictionaries
- NumPy arrays
- Most user defined types (classes)

Examples of immutable objects in Python:
- Strings
- Tuples
## Scalar Types

A scalar type is is a single value type muilt-in for handlin numerical data, strings, Boolean calues, dates and time.

### Strings

#### count
count the amount of times a string or character appears in a string. For example:

```Python
c = "Esa campaña trajo honores y promoción para muchos, pero para mí trajo desgracia y desastre, pues me sacaron de mi brigada y me incorporaron a la de Berkshire"

c.count("a")
```
Produces the ouptut:
```Shell
20
```

#### replace
Since Python strings are inmutable, so if we need to modify a string, we have to use the `replace` method, which creates a new string and leaves the original one unmodified. For example:
```Python
b = c.replace("a", "A")
b
```
Creates the output:
```Shell
'EsA cAmpAñA trAjo honores y promoción pArA muchos, pero pArA mí trAjo desgrAciA y desAstre, pues me sAcAron de mi brigAdA y me incorporAron A lA de Berkshire'
```
#### String templating or formatting
##### format
This method can be used to substitute formatted arguments into a string, for example:

```Python
template = "{0:.2f} {1:s} are worth US${2:d}"
template.format(86.99, "Mexican pesos", 5)
```
Produces the output:
```Shell
'86.99 Mexican pesos are worth US$5'
```
##### f-string
It is another way to create formatted strings. To create an f-string we write de character `f` immediately preceding a string literal. Withing the string we substitute the value of the expression.

```Python
amount = 10
rate= 88.46
currency = "Pesos"

result = f"{amount} {currency} is worth US${amount/rate:.2f}"

result
```
Output:
```shell
'10 Pesos is worth US$0.11'
```

### Bytes and Unicode
**Unicode** has become the first-class string type to enable more consistent handling of ASCII and non-ASCII text, by assigning a unique number to every letter, symbol and emoji in the world, regardless of the language or device. 

**UTF-8** is the specific system used to translate those Unicode numbers into bytes that a computer can store and send. It uses **1 byte** for basic English letters, **2 to 3 bytes** for accented letters or other alphabets, and **4 bytes** for emojis.

![[Pasted image 20260721223620.png]]

#### encode
We use it to convert a Unicode string to its UTF-8 bytes representation.
```Python
val = "español" #unicode
print(val)

val_utf8 = val.encode("utf-8") #to UTF-8
val_utf8
```
Produces the output:
```shell
español
b'espa\xc3\xb1ol'
```

#### decode
We use it to convert a UTF-8 bytes object to its Unicode string representation.
```Python
val_utf8.decode("utf-8") #to Unicode
```
Output:
```shell
'español'
```

### Dates and times

#### Format specifications
![[Pasted image 20260721230241.png]]
![[Pasted image 20260721230408.png]]

`datetime` module provides `datetime`, `date`, and `time` types. `datetime` combines `date` and `time` that can be used as well to extract the same object from `datetime`.
```Python
from datetime import datetime, date, time

dt = datetime(2026,7,21,22,47,23)

print(dt.minute, dt.second)
print(dt.date())
print(dt.time())
```
Output:
```shell
47 23
2026-07-21
22:47:23
```
#### strftime

formats a `datetime` as a string:
```Python
dt.strftime("%d-%m-%Y %H:%M")
```
Output:
```shell
'21-07-2026   22:47'
```

#### strptime

Turns a string into `datetime` objects.
```Python
datetime.strptime("20010627","%Y%m%d") #strptime(string,format)
```
Output:
```shell
datetime.datetime(2001, 6, 27, 0, 0)
```
#### replace (datetime)
Since `datetime` object is immutable, we must use this function, which creates a new, modified object, to change its values:
```Python
dt_hour = dt.replace(minute = 0, second = 0)
print(dt_hour)
dt_hour
```
Output:
```shell
2026-07-21 22:00:00

datetime.datetime(2026, 7, 21, 22, 0)
```

#### timedelta and operations with `datetime`

The difference between two dates is calculated by a simple subtraction, such as:

```Python
dt2 = datetime(2011, 11, 15, 22, 30)
delta = dt2 - dt
delta
```

Output:
```shell
datetime.timedelta(days=-5363, seconds=85357)
```

Then, you can add a `timedelta` to a `datetime` to produce a shifted `datetime`:
```Python
dt2 + delta
```
Output:
```shell
datetime.datetime(1997, 3, 11, 22, 12, 37)
```
